```
> 帮我看看微信最近的会话

```

  文本: 小李: 文档我下午给到大家

## 4. 时间范围 + 分页查看聊天记录

```
> 帮我看一下读书会2026 3 月 1 日到 3 月 7 日的聊天，先给我前 20 条
```

Claude 可以调用：

```python
get_chat_history(
    chat_name="读书会2026",
    start_time="2026-03-01",
    end_time="2026-03-07",
    limit=20,
    offset=0,
)
```

下一页：

```python
get_chat_history(
    chat_name="读书会2026",
    start_time="2026-03-01",
    end_time="2026-03-07",
    limit=20,
    offset=20,
)
```

## 5. 搜索指定联系人/群聊在某个时间段内的消息

```
> 帮我搜一下项目A组这周谁提到过"截止日期"
```

Claude 可以调用统一接口：

```python
search_messages(
    keyword="截止日期",
    chat_name="项目A组",
    start_time="2026-03-01",
    end_time="2026-03-07",
    limit=20,
    offset=0,
)
```

## 6. 多个联系人/群聊联合搜索

```
> 帮我看看小李、小王 和 项目A组 这周谁提到过"上线"
```

Claude 可以调用统一接口：

```python
search_messages(
    keyword="上线",
    chat_name=["小李", "小王", "项目A组"],
    start_time="2026-03-01",
    end_time="2026-03-07",
    limit=20,
    offset=0,
)
```

如果某些名字没匹配到联系人，或没有对应消息表，结果里会单独说明。

## 9. 高级用法：群聊分析

Claude 可以获取大量消息后自动分析活跃度、话题分布、关键人物：

```
> 帮我分析一下读书会2026最近一周的情况
```

Claude 会调用 `get_chat_history(chat_name="读书会2026", limit=500)` 获取消息，然后输出。消息很多时，也可以把 `limit` 设得更大，或配合 `offset` 分页读取：

```
## 读书会2026 最近一周分析

### 话题分布
| 话题     | 占比  |
|----------|-------|
| 章节讨论 | ~50%  |
| 阅读心得 | ~25%  |
| 下次安排 | ~15%  |
| 闲聊     | ~10%  |

### 活跃度排名
| 排名 | 昵称 | 发言数 | 角色       |
|------|------|--------|------------|
| 1    | 小王 | ~80条  | 章节带读者 |
| 2    | 老李 | ~50条  | 深度讨论   |
| 3    | 小张 | ~30条  | 新人提问   |

### 群氛围
节奏稳定，本周第3章讨论参与度最高，下周进入第4章。
```
